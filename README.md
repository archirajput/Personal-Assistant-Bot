# 🤖 Archie - Your Personal Assistant

An AI-powered personal assistant chatbot built with Flask and Google Gemini API. Archie can answer questions, assist with tasks and hold intelligent conversations.

## Features
- Conversational AI powered by Google Gemini
- Clean and modern chat interface
- Real-time responses
- Built with Flask backend

## Tech Stack
- Python, Flask
- Google Gemini API
- HTML5, CSS3, JavaScript

## How to Run
```bash
git clone https://github.com/archirajput/Personal-Assistant-Bot.git
cd Personal-Assistant-Bot
pip install flask google-generativeai
python app.py
Then open http://127.0.0.1:5000 in your browser.
